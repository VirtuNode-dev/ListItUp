# ListItUp Docs

This folder stores durable project knowledge: questions, planned specs, completed specs, architecture decisions, templates, and agent configuration.

## Folders

- `QnA/`: grilling sessions from `grill-with-docs`, including the questions, recommended answers, user answers, and settled outcomes.
- `Specs-Planned/`: PRDs and specs that are planned or in progress.
- `Specs-Completed/`: specs that have shipped or are no longer active.
- `ADR/`: architecture decision records.
- `Templates/`: reusable doc templates.
- `agents/`: configuration read by AI agent skills.

## Operating Model

Use `grill-with-docs` to sharpen fuzzy ideas and store the session in `QnA/`, `to-prd` to produce a PRD, `to-issues` to split it into vertical slices, and `tdd` to implement each slice test-first.
