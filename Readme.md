# ListItUp

ListItUp is being set up as an AI-buildable product repo with durable project context in markdown.

Start here:

- `AGENTS.md` for agent workflow and repo rules.
- `CONTEXT.md` for product language.
- `Brand.md` for brand voice and positioning.
- `DESIGN.md` for interface direction.
- `sample.html` for the current visual reference implementation.
- `docs/` for QnA, specs, ADRs, templates, and agent configuration.
